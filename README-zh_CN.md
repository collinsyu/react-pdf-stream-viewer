# 一个pdf展示组件
>日拱一卒无有尽,功不唐捐终入海

支持流pdf文件预览 和 静态pdf文件预览

本组件最大亮点是支持 ~流~pdf文件预览，而且加上了 类似chrome 默认pdf的功能按钮样式



# 功能
1. 支持流pdf文件预览
2. ui和chrome默认ui一致
